Documentation

Learn how to install and use FrameForge to track your Warframe inventory in real time.

Installation

System requirements

  • Windows 10 or Windows 11 (64-bit)
  • Warframe — Steam or standalone client
  • No administrator privileges required at runtime

Steps

  1. Go to the latest release on GitHub.
  2. Download FrameForge_1.0.1_x64-setup.exe.
  3. Run the installer and follow the prompts.
  4. Launch FrameForge from the Start menu or desktop shortcut.
SmartScreen warning: Windows may warn that the app is from an unknown publisher. This is because code-signing certificates cost ~$300/year and FrameForge is a free, open-source project. Click More info → Run anyway to proceed. The full source code is on GitHub if you want to verify what you're running.

First Launch

You can launch FrameForge before or after starting Warframe — it doesn't matter which order. The app polls for a running Warframe process at startup and will detect it automatically.

Triggering an inventory refresh

Return to your Orbiter after completing a mission or visiting a relay. Your inventory will refresh within about 10 seconds.

Heads up: Opening the in-game ESC menu does not reliably trigger a scan. Returning to your Orbiter is the most consistent refresh trigger.

Inventory

The Inventory tab shows everything tracked in your Warframe account, including:

  • Crafting resources (Nano Spores, Morphics, Plastids, Polymer Bundle, etc.)
  • Blueprints and recipe items
  • Warframes, weapons, and companions (unique items)

Quantities are updated automatically whenever FrameForge detects a valid inventory state in memory — typically when you return to your Orbiter.

How scanning works

FrameForge uses two scanners that run in parallel:

  • Resource scanner — finds crafting resources and blueprints by matching the ItemCount / ItemType pattern in memory.
  • Unique item scanner — locates warframes, weapons, and companions using a fast Aho-Corasick pattern match on known item paths.

Neither scanner writes to memory or modifies game files.

Foundry

The Foundry tab helps you plan builds. Items are organized into categories:

  • Warframes
  • Primary weapons
  • Secondary weapons
  • Melee weapons
  • Companions
  • Archwing

Component status colours

  • Gold — you own this component or already have it built.
  • Blue — you have the blueprint but not the component itself.
  • Dark — you're missing both the component and the blueprint.

Relic indicators

If a missing component can be obtained from a relic you currently own, a relic icon appears next to it. This makes it easy to prioritise which relics to run.

Tracking panel

Click any item to open a tracking panel showing every required component and its status. Use the Missing only toggle to focus on what you still need.

OCR Reward Overlay

When a void fissure reward screen opens in Warframe, FrameForge automatically detects it and displays a small overlay showing each of the four reward cards with:

  • Item name
  • Current platinum sell price (from warframe.market)
  • Ducat value
  • Set completion status — whether you need it for a build

This lets you pick the best reward instantly without alt-tabbing out of the game.

How it works

  1. FrameForge monitors EE.log for the void reward screen marker.
  2. After a short delay for the card fade-in animation, it captures the top portion of the Warframe window.
  3. Windows OCR reads the text from each card slot.
  4. Rarity bar colour analysis classifies each reward as Bronze / Silver / Gold / Platinum.
  5. The overlay appears with prices and ducat values ranked by value.
Best results: The overlay works in Windowed and Borderless Windowed mode. Fullscreen Exclusive is also supported via DXGI capture.

Market Helper

The Market Helper lets you look up live platinum sell prices for any item or prime set, sourced from the public warframe.market v2 API.

  • Prices are fetched on demand — no account required.
  • No data about you is sent anywhere; requests go directly from your machine to warframe.market (the same as visiting the website in a browser).

Relic Helper

The Relic Helper lets you browse void fissure drop tables and work out the most efficient relics to run for prime parts you're still missing.

  • Drop rate data comes from community-maintained sources.
  • Cross-referenced with your current inventory so you can focus only on parts you don't have.

Mastery Rank

Your current Mastery Rank is displayed as a gold badge in the app header. It's read from the PlayerLevel field in game memory during each scan cycle, so it always reflects your live rank.

Vault Status

Every prime item in the Foundry shows a vault badge:

  • 🔒 Vaulted — parts are currently unavailable in the standard relic drop table.
  • 🔓 Available — parts can be obtained from active relics right now.

Vault data comes from the WFCD warframe-items dataset and is bundled with each FrameForge release.

FAQ

Is FrameForge safe to use?

FrameForge is strictly read-only. It uses ReadProcessMemory — the same Windows API used by Overwolf, Discord overlay, and hardware monitors. It never writes to memory, injects code, or hooks the game in any way.

Network requests go only to warframe.market (for prices) and community GitHub repos (item catalog, drop rates). There is no FrameForge server, no telemetry, and nothing leaves your machine beyond those two sources.

Digital Extremes has publicly confirmed that read-only memory tools are acceptable. The full source code is available on GitHub under GPLv3 — you can read every line and verify exactly what it does.

Why doesn't my inventory update when I press ESC?

The in-game ESC menu doesn't reliably put inventory data into memory in a scannable state. Return to your Orbiter after a mission or relay visit for the most consistent results.

Will you add mods or arcanes?

Not currently. Relay data contaminates small mod counts, making accurate tracking unreliable. It may be revisited in a future version.

Will you add trading or market features?

Not planned. FrameForge focuses on inventory and crafting. For trading, check out warframe.market.

Can I use FrameForge on Linux or Mac?

No. FrameForge uses Windows-specific APIs and is only supported on Windows 10 / 11.

How do I report a bug?

Open an issue on the GitHub Issues page. Include your Windows version and a description of what happened.

Known Limitations

  • Mods and arcanes — not tracked; relay data contaminates small counts.
  • Mission rewards — in-mission reward format is not yet identified; only post-mission inventory changes are captured.
  • Rivens — not tracked.
  • Sold item mastery — mastery from sold items would require account API authentication, which is not implemented.
  • Slow first scan — the first scan after launch may take up to 90 seconds on debug builds. Release builds are significantly faster.
